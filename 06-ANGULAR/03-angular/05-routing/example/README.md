# 🧭 BÀI 5 — ROUTING TRONG ANGULAR 17 (STANDALONE STYLE)

- RouterOutlet:
  - Hiển thị trang con bên trong trang chính
- RouterLink
  - Chuyển trang trong SPA (không reload)
- Routes + provideRouter()
  - Cấu hình định tuyến
- loadComponent()

  - Lazy load component — cú pháp mới Angular 17

- @defer

  - Chờ (lazy render) component sau khi tải

- `⚙️ 1️⃣ Cấu trúc routing trong Angular 17+`

- Trong Angular mới, routing được định nghĩa trong file `app.routes.ts,` và khởi tạo trong app.config.ts (thay vì AppModule như ngày xưa).
